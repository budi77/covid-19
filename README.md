

## COVID-19 STATS

Sistem yang akan memaparkan statistik pandemik Virus Corona (Covid-19) di seluruh dunia. Data adalah diambil daripada beberapa sumber menggunakan API.

## MAKLUMAT SISTEM

- Laravel 7.3
- ChartJs
- Livewire
- Public API
